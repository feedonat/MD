About This App

This is an example page demonstrating the reusable header component.

HeaderComponent Features:

  • Standalone component - can be used anywhere
  • Accepts title input for dynamic titles
  • Accepts translucent input (this page has it disabled)
  • Integrates seamlessly with TitleService

TitleService Features:

  • Manages page titles using Angular signals
  • Reactive updates across components
  • Simple API: setTitle() and getTitle()
